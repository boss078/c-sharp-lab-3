В папке можно найти батники install_it и delete_it, которые нужны для установки/удаления службы(запускать с админ правами). Далее службу Service1 нужно запустить через менюшку винды. После этого можно тестовую папку поместить в Source directory и наблюдать за работой проги. Выбирать, какой конфиг использовать, xml или json, можно только в вижле с перекомпиляцией xD
